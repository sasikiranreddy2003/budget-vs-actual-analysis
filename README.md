# 📊 Budget vs Actual Analysis – FP&A Portfolio Project

## Overview

This project demonstrates a Budget vs Actual analysis commonly performed by Financial Planning & Analysis (FP&A) teams. It compares budgeted financial performance with actual results, calculates variances, and presents key insights through an executive dashboard.

---

## Business Objective

The objective of this project is to:

- Compare budgeted and actual financial performance
- Identify favourable and unfavourable variances
- Analyse departmental spending
- Monitor monthly revenue and operating expenses
- Support management decision-making using financial reporting

---

## Key Calculations

**Variance (£)**

```
Actual - Budget
```

**Variance (%)**

```
Variance ÷ Budget
```

**Operating Profit**

```
Revenue - Operating Expenses
```
